## Project Configuration

### Package Manager
**CRITICAL: This project uses `pnpm`, NOT `npm`.**

Always use:
- `pnpm install` (not npm install)
- `pnpm run dev` (not npm run dev)
- `pnpm run build` (not npm run build)
- `pnpm add <package>` (not npm install)
